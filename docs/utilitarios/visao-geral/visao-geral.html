Visão Geral

Utilitários CSS e JavaScript são trechos de código pré-definidos que fornecem pequenas funcionalidades para uso em projetos web. Tais recursos permitem que os desenvolvedores apliquem rapidamente estilos e efeitos dinâmicos em seus projetos, sem a necessidade de escrever código personalizado para cada elemento.


Utilitários CSS

Os utilitários CSS trazem classes que definem estilos e disposição dos elementos na interface, como margens, paddings, posicionamento de elementos, tamanhos de fonte, cores de superfícies e muitos outros.


Tipografia

Os utilitários de Tipografia foram criados com base no Fundamento Tipografia e aplicam os estilos para os tipos de textos, como: títulos, label, legend, input, etc.

Overflow

O utilitário overflow é usado para definir como o conteúdo se relaciona com o elemento que o contém.

Flexbox

É usado para controlar a disposição dos elementos na interface.

Grid

Elas servem para delimitar a largura do conteúdo de acordo com os breakpoints ou resolução de tela.

Display

Define como um elemento HTML deve ser exibido na página.

Espaçamento

Usados para definir as distâncias ou espaçamentos entre elementos da interface. Abordam aspectos como: breakpoints, tamanhos de escala, direção e sentido.

Cores

São baseadas no Fundamento de Cores e suas funções de aplicação, como: cor de superfície, leitura, borda e estados.

Textos

Foram criadas com base no Fundamento Tipografia e definem propriedades como: peso, tamanho, alinhamento, transformação, etc;

Bordas

Tem sua base no Fundamento Superfície e tratam propriedades como: espessura ou tipo de bordas.

Arredondamento

Baseadas no Fundamento Superfície, tratam propriedades de arredondamento.

Elevação

Criado com base no Fundamento Elevação tratam as propriedade de sombreamento dos elementos.

Movimento

Tem sua base no Fundamento de Movimento e tratam as propriedades de Easing e Duração.

Utilitários JavaScript

Os utilitários JavaScript fornecem funcionalidades dinâmicas para interface, como efeitos de animação, repostas às interações de usuários, rotinas automáticas, chamadas de funções, entre outros comportamentos diversos.


Accordion

Tratam comportamentos e atributos do elemento acionador e elemento alvo do accordion.

Checkgroup

Abordam comportamentos do agrupamento de checkboxes, como: atributos do checkbox pai, atributos do checkbox filho, estado indeterminado e checkgroup em cadeia.

Collapse

Tratam comportamentos para o padrão Collapse como: atributos para o elemento acionador e elemento alvo.

Dropdown

Tratam comportamentos para o padrão Dropdown como: atributos para o elemento acionador e elemento alvo.

Scrim

Abordam comportamentos para o uso do Scrim.

Tooltip

Abordam comportamentos para o Tooltip como: atributos do Tooltip com texto ou Tooltip com elementos.
