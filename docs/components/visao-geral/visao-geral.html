Os componentes são elementos de interface web oferecidos pelo Padrão Digital de Governo. São projetados usando tecnologia agnóstica com as linguagens HTML, CSS e JavaScript e encapsulam funcionalidades complexas em unidades independentes. Uma das principais características dos componentes é o poder de reutilização. São consistentes e fáceis de manter, pois, uma vez criados, eles podem ser facilmente reutilizados em diferentes partes de um sistema ou página web. Eles permitem que desenvolvedores construam sites e aplicativos mais rapidamente, sem a necessidade de recriar códigos ou conceitos de design já estabelecidos. Além disso, os componentes já trazem em sua estrutura padrões testados para garantir uma boa acessibilidade.

Por fim, os componentes já foram desenvolvidos considerando o padrão visual e a experiência única oferecidos pelo design system. Dessa forma, os usuários conseguem facilmente associar as características visuais dos produtos e serviços desenvolvidos pelo Governo Federal.


Avatar

Componente utilizado para representar graficamente um usuário em ambientes virtuais. Pode apresentar-se como ilustração vetorial (avatar icônico), texto (avatar letra) e fotografia (avatar fotográfico).

Breadcrumb

Breadcrumb é um sistema de navegação estrutural usado para proporcionar ao usuário um meio de localização dentro da estrutura de navegação do site ou sistema.

Button

Os botões são elementos interativos da interface, que permitem que os usuários acessem funcionalidades, executem ações ou naveguem pela interface.

Card

Cards são cartões (ou superfícies) que contêm conteúdo e ações diversas sobre um único assunto (ou tópico).

Carousel

Carousel é um componente que permite que vários conteúdos ocupem um único espaço na tela e que o usuário navegue por eles de forma sequencial e controlada.

Checkbox

O componente checkbox permite selecionar uma ou mais opções em uma lista de opções.

CookieBar

O cookieBar é o componente responsável por fazer a gestão de cookies de um determinado produto, ele deve ser utilizado quando houver a necessidade de fazer qualquer tipo de tratamento de dados do usuário por meio de cookies.

DateTimePicker

DateTimePicker é um componente que auxilia o usuário na seleção de uma data a partir de um calendário e/ou uma hora a partir de um seletor de horas.

Divider

Divider, também conhecido como Divisor ou Separador, é uma linha neutra que separa conteúdos, sessões ou temas afins.

Footer

Geralmente localizado na parte inferior das páginas, o footer (rodapé) pode ser organizado de formas distintas.

Header

O componente Header é o principal elemento de uma página, onde podem ser agrupados componentes predefinidos que tem como finalidade auxiliar o usuário no acesso ou mesmo execução das funcionalidades principais site/sistema.

Input

Componente que permite a entrada de dados textuais por parte do usuário.

Item

Itens são blocos compactos e altamente flexível diagramados para exibir uma ampla variedade de conteúdo de forma repetida.

List

Lists são estruturas compostas por itens em sequência vertical ou horizontal que podem conter imagens, ícones e elementos interativos.

Loading

São indicadores de progresso que expressam um tempo de espera não especificado ou exibem a duração de um processo.

Magic Button

O componente Magic Button é um botão com grande apelo visual.

Menu

Os menus de navegação (ou simplesmente menus) são listas de categorias de conteúdo ou recursos, normalmente apresentados como um conjunto de links ou ícones agrupados com um estilo visual diferente do resto do design.

Message

O Message (Mensagem) é um Componente de Interface que tem como finalidade proporcionar feedback ao usuário sobre o que ocorre no sistema.

Modal

Modal é uma janela que exibe um conteúdo adicional em uma camada acima da página atual, com uma sobreposição de superfície (overlay) cobrindo a página e tornando-a temporariamente inacessível.

Notification

Notification (notificação) são mensagens enviadas por aplicativos ou sites que fornecem informações curtas, oportunas e relevantes ao usuário.

Pagination

O Pagination é um Componente de interface que tem a função de organizar conteúdo de dados em páginas sequenciais, trazendo maior usabilidade durante o consumo da informação pelo usuário.

Radio

O componente Radio permite selecionar apenas uma opção em uma lista de opções.

Select

O componente select permite selecionar itens em uma lista de opções.

Sign-in

Sign-in é um botão que envia um ID de login do usuário e senha ao servidor para autenticação e é utilizado para direcionar o usuário a uma área de confirmação de acesso que utiliza dados do usuário previamente cadastrados.

Skip Link

O componente skip link consiste em uma lista de itens que auxiliam os usuários a navegarem por uma página com o auxílio do teclado, sendo extremamente útil ou até mesmo indispensável para aqueles que dependem do teclado para a navegação.

Step

Steps são indicadores de etapas e progresso que auxiliam os usuários durante um fluxo de trabalho, seguindo uma sequência lógica linear, aleatória, numeradas ou não.

Switch

Switches são controles usados para alternar rapidamente entre dois estados possíveis. São usados ​​apenas para ações binárias que ocorrem imediatamente após o usuário “girar” o botão de alternância. São comumente usados ​​para interruptores “liga/desliga”.

Tab

As Tabs (Abas) são Componentes de Navegação em Interface que tem a função de organizar o conteúdo da página em categorias ou seções, oferecendo maior usabilidade ao usuário.

Table

A Table (Tabela) é um conjunto de dados organizados em linhas e colunas.

Tag

Tag é qualquer tipo de sinalizador que pode indicar um status, uma quantidade ou uma informação de forma precisa e direta.

Textarea

O Textarea é um campo de entrada de texto que permite que seu aplicativo aceite um valor de texto potencialmente longo do usuário.

Tooltip

São alertas flutuantes que proporcionam ao usuário um feedback do sistema, informando sobre algum aspecto da interface que mereça maior atenção.

Upload

Upload é um componente que permite ao usuário selecionar um ou mais arquivos e os enviar por meio de um servidor de aplicação.

Wizard

Wizard é um processo passo a passo que permite ao usuário inserir informações em uma ordem prescrita e na qual as etapas subsequentes podem depender das informações inseridas nas etapas anteriores.
