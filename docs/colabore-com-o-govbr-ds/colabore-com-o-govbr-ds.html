Colabore com o GOVBR-DS

O time GOVBR-DS acredita que para um produto crescer de forma consistente e alinhada com seus usuários é preciso estar aberto a comentários, sugestões e críticas. Ninguém melhor que você para nos dizer como podemos realizar entregas mais eficientes. Por isso, gostaríamos de ouvir o que você tem para nos contar e juntos fazermos um Design System para todos.

Como Colaborar?

Qualquer feedback é importante para nós, pois só assim podemos saber onde estamos errando e acertando, desta forma aprimorarmos nossas entregas.

Gostaríamos de saber coisas do tipo:

- Quais componentes você gostaria de encontrar nas próximas versões do GOVBR-DS?

- O que podemos fazer para melhorar os componentes que já oferecemos?

- Como o site do GOVBR-DS pode ser mais fácil e objetivo?

- O que fazer para estarmos ainda mais próximos de você?

Enfim, tudo o que pode ser melhorado nos interessa.
